# AI抖音选题下载

输入一段脚本后，动作会调用 OpenAI 兼容的第三方 AI 接口生成抖音搜索关键词，并进入以下流程：

1. 多选 AI 返回的搜索关键词。
2. 调用本机浏览器扩展抓取抖音搜索结果。
3. 以封面图片网格展示视频结果。
4. 多选需要的视频封面。
5. 确认后下载选中的视频到指定目录，并保存对应元数据 JSON。

## 使用前准备

- 已安装并加载 `插件/抖音关键词搜索扩展`。
- 首次运行时填写 API 地址、API 密钥、模型 ID、系统提示词、脚本内容、下载目录。
- API 密钥只在本地 Quicker 变量中保存，不建议写入分享版动作配置。

## 默认接口

- API 地址：`https://new.12ai.org/v1/chat/completions`
- 模型 ID：`gemini-3.1-flash-lite-preview`

## 提示词要求

系统提示词可以自由修改。建议要求 AI 只返回 JSON 字符串数组，例如：

```json
["糖尿病饮食", "血糖控制", "胰岛素抵抗"]
```
