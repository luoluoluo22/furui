# 中医药信息查询

输入关键词并选择分类，动作会访问大医网搜索页，打开完全匹配的搜索结果，并提取对应栏目内容。

## 支持分类

- 疾病：搜索 `type=disease`，匹配 `关键词 - 疾病`，提取详情页“鉴别诊断”小标题。
- 药品：搜索 `type=medical`，匹配 `关键词 - 西药/中成药`，提取详情页“适应症”内容。
- 中药材：搜索 `type=cmedical`，匹配 `关键词 - 中药材`，提取详情页“加工炮制 > 炮制方法”的第一条，并去掉编号和药材名前缀。

## 界面功能

- 输入关键词。
- 选择分类：疾病、药品、中药材。
- “历史”下拉框会保留成功查询过的分类、关键词和提取内容预览，最多 50 条。
- “模板配置”可分别配置疾病、药品、中药材三个输出模板。
- 查询结果窗口可直接编辑最终结果。
- “复制结果”按钮复制最终结果。
- “原始内容”按钮显示提取到的原始栏目内容，并支持复制原文。

## 输出变量

- `rtn` / `text`：最终格式化结果。
- `rawContent`：原始提取内容。
- `searchUrl`：搜索页地址。
- `detailUrl`：详情页地址。
- `matchedTitle`：命中的标题。
- `history`：查询历史记录，会本地保存分类、关键词和提取内容。
- `keyword` / `category`：本次查询参数，会本地保存。

## 输出模板

可在搜索界面点击“模板配置”控制最终输出。三个模板会分别保存到以下变量：

- `diseaseFormat`：疾病模板
- `medicalFormat`：药品模板
- `cmedicalFormat`：中药材模板

默认模板：

- 疾病：`{keyword}鉴别诊断：需与{items}相鉴别`
- 药品：`{keyword}适应症：{content}`
- 中药材：`{keyword}炮制：{content}`

可用占位符：

- `{keyword}`：输入关键词
- `{category}`：分类名称
- `{title}`：命中的标题
- `{field_name}`：提取栏目名
- `{content}` / `{raw}`：最终选中的内容
- `{raw_content}`：原始栏目内容
- `{items}`：疾病鉴别诊断小标题，用顿号连接
- `{items_with_index}`：疾病鉴别诊断带编号列表
- `{search_url}`：搜索页地址
- `{detail_url}`：详情页地址
- `{newline}`：换行

## 示例

疾病，关键词：糖尿病

`糖尿病鉴别诊断：需与肾性糖尿、应激性高血糖、其他原因所致尿糖阳性相鉴别`

药品，关键词：西地那非

`西地那非适应症：本品适用于治疗勃起功能障碍。`

中药材，关键词：山药

`山药炮制：取原药材，除去杂质，大小条分开，浸泡3-4成透，捞出，闷润至透，切厚片，及时干燥。`
